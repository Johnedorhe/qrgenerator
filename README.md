# QR Code Generator

A simple web application to generate QR codes from user input using vanilla JavaScript.

## Features

- Enter any text or URL to generate a QR code
- Click the "Generate QR Code" button or press Enter
- Responsive design for mobile and desktop

## How to Use

1. Open `index.html` in your browser.
2. Type your text or URL in the input field.
3. Click the "Generate QR Code" button or press Enter.
4. The QR code will appear below.

## Files

- `index.html`: Main HTML file
- `main.js`: JavaScript logic for QR code generation
- `style.css`: Styling and responsive design

## Dependencies

- [QRCode.js](https://github.com/davidshimjs/qrcodejs) (loaded via CDN)
